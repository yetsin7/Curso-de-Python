## ¿Que es Python?

Python es un lenguaje de programacion. Sirve para escribir instrucciones que una
computadora pueda seguir.

Es muy usado para aprender porque su sintaxis es legible, pero tambien se usa en
trabajo real: automatizacion, APIs, testing, analisis de datos, IA y scripts.

---

## ¿Que es un programa?

Un programa es una secuencia de instrucciones.

Por ejemplo:

1. pedir un nombre;
2. guardar ese dato;
3. mostrar un saludo.

Eso ya es logica de programacion. Python solo nos da una forma clara de
escribirla.

---

## ¿Que pasa dentro de la computadora?

Cuando ejecutas un archivo Python:

1. el sistema operativo abre el interprete;
2. Python lee tu archivo;
3. analiza la sintaxis;
4. crea datos en memoria;
5. ejecuta instrucciones en orden;
6. muestra resultados en la consola.

La consola es la primera ventana para observar el comportamiento del programa.

## Conceptos clave

- **`print()`**: muestra datos en la consola.
- **Variable**: nombre que referencia un valor en memoria.
- **Tipo de dato**: clase de valor con la que trabaja Python.
- **`input()`**: pide texto al usuario desde teclado.
- **Comentario**: ayuda para humanos que Python no ejecuta.

---

## Errores comunes del principiante

- creer que programar es memorizar lineas;
- olvidar ejecutar y observar la salida;
- confundir texto con numeros;
- no entender que `input()` al inicio devuelve texto;
- saltarse comentarios que explican el contexto.
