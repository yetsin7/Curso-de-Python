# Documentacion oficial de Python

Python tiene una de las mejores documentaciones del mundo. Este archivo te deja
los enlaces mas importantes para acompanar el curso.

## Referencias principales

- [Python documentation](https://docs.python.org/3/)
- [Python tutorial](https://docs.python.org/3/tutorial/)
- [Python standard library](https://docs.python.org/3/library/)

## Recursos clave para estudiar

- [Tipos de datos](https://docs.python.org/3/library/stdtypes.html)
- [Control de flujo](https://docs.python.org/3/tutorial/controlflow.html)
- [Funciones](https://docs.python.org/3/tutorial/controlflow.html#defining-functions)
- [Manejo de archivos](https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files)
- [Excepciones](https://docs.python.org/3/tutorial/errors.html)
- [PEP 8](https://peps.python.org/pep-0008/)

## Orden recomendado

1. Lee el tutorial oficial desde fundamentos.
2. Refuerza tipos, funciones y estructuras.
3. Usa la libreria estandar como referencia diaria.
4. Apoya tu practica con PEP 8 para escribir mejor codigo.
