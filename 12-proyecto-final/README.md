# Capítulo 12 — Proyecto Final: Gestor de Tareas

¡Llegaste al final del libro! Este capítulo integra todo lo que aprendiste
en un proyecto real y funcional: un **gestor de tareas en consola**.

---

## ¿Qué construirás?

Una aplicación de línea de comandos (CLI) para gestionar tareas personales.
Funciona completamente offline y guarda los datos en un archivo JSON local.

### Funcionalidades
- Agregar tareas con título, descripción y prioridad
- Listar todas las tareas (con filtros)
- Marcar tareas como completadas
- Eliminar tareas
- Guardar y cargar automáticamente desde archivo
- Estadísticas del estado de las tareas

---

## Conceptos que usa este proyecto

| Concepto | Dónde se usa |
|---|---|
| Variables y tipos de datos | En toda la aplicación |
| Listas y diccionarios | Para almacenar las tareas |
| Funciones | Cada funcionalidad es una función |
| Clases (POO) | `Tarea` y `GestorDeTareas` |
| Manejo de errores | Al leer/escribir el archivo |
| Archivos JSON | Para persistir las tareas |
| Módulos (`json`, `os`, `datetime`) | Importados al inicio |
| f-strings | Para mostrar la información |
| Comprensiones | Para filtrar tareas |

---

## Archivos del proyecto

| Archivo | Descripción |
|---|---|
| `gestor_tareas.py` | El programa completo, listo para ejecutar |

---

## ¿Cómo ejecutarlo?

Abre una terminal en esta carpeta y ejecuta:
```
python gestor_tareas.py
```

O desde VS Code: abre `gestor_tareas.py` y presiona `Ctrl+F5`.
