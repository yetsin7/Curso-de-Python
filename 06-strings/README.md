Ya conoces los strings desde el capítulo 1. Ahora aprenderás todo su potencial.

increíblemente útiles en aplicaciones reales.

---

## ¿Qué es un string?

Un string es una secuencia de caracteres. Cada letra, número, espacio o símbolo
es un carácter. Python los almacena internamente como una secuencia indexada,
igual que una lista (pero inmutable).

```python
texto = "Python"
#         P y t h o n
# índice: 0 1 2 3 4 5
```

---

## ¿Por qué es importante dominar los strings?

Casi toda aplicación del mundo real trabaja con texto:
- Nombres de usuarios, contraseñas, emails
- Mensajes, notificaciones, reportes
- Leer y escribir archivos
- Procesar datos de APIs y bases de datos
